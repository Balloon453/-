HTML وCSS من الصفر حتى الاحتراف

نُشر بتاريخ: 20 مايو 2025

تعلم أساسيات بناء صفحات الويب باستخدام HTML وCSS، بداية من إنشاء الهيكل الأساسي حتى تصميم الصفحات بشكل احترافي.

مقدمة في HTML

HTML هي لغة ترميز تستخدم لبناء هيكل صفحات الويب.

تصميم الصفحات بـ CSS

CSS تستخدم لتنسيق وتصميم صفحات الويب، مثل الألوان والخطوط والتخطيط.

معاً ستتعلم كيف تبني موقع ويب متكامل من البداية حتى النهاية.
